# Assignment 4. Опрацювання файлів. Бібліотеки Numpy, Pandas, Matplotlib

## Мета
Навчитися зчитувати дані з файлів формату CSV, обробляти їх за допомогою векторних операцій, організовувати у таблиці (DataFrame), зберігати результати у JSON та візуалізувати математичні моделі. Продемонструвати навички роботи з числовими масивами та табличними даними під час дослідження моделі випадкового блукання.

У сучасній прикладній математиці аналітичне розв'язання задач часто супроводжується чисельним моделюванням. Вміння швидко завантажити масив експериментальних даних, ефективно обробити його та презентувати у вигляді зрозумілого графіка — це ключова навичка для математичного моделювання та аналізу.

## Завдання
1. Зробіть форк репозиторію викладача (https://github.com/yKozolup/CS201_Assignment4) та склонуйте його, щоб отримати доступ до файлу `random_walk.csv`, який містить координати двовимірного випадкового блукання (колонки `step`, `x`, `y`).
2. Зчитайте дані з цього файлу та перетворіть їх у формат таблиці DataFrame.
3. Використовуючи векторні математичні операції, обчисліть евклідову відстань від початкової точки (0, 0) для кожного кроку за формулою $D=\sqrt{x^{2}+y^{2}}$.
4. Додайте обчислені відстані як нову колонку `distance` до вашої таблиці.
5. Проведіть розрахунки, корисні для аналізу моделі: знайдіть максимальну та середню відстань від центру за весь час блукання та виведіть на екран.
6. Виведіть таблицю лише з тими кроками, де відстань до центру є більшою, ніж середнє значення (точки значних відхилень).
7. Збережіть ці відфільтровані дані у новий файл з назвою `filtered_walk.json`.
8. Створіть лінійний графік, який відображає траєкторію випадкового блукання (залежність координати `y` від `x`).
9. Позначте початкову точку (0, 0) та кінцеву точку різними кольорами або маркерами, щоб було зрозуміло, де блукання почалося і де завершилося.
10. Додайте заголовки, підписи осей, легенду, та змініть кольори за замовчуванням на ті, які вважаєте необхідними.
11. Створіть Pull Request з вашого форку до оригінального репозиторію викладача та надішліть посилання на нього у відповідне завдання на платформі курсу.
